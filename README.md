# Wildberries Price Parser

Парсер цен с [Wildberries.ru](https://wildberries.ru) — CLI-инструмент и Telegram-бот.  
Введи название товара и получи цены, рейтинг и ссылки.

---

## Telegram Bot

Самый удобный способ использования. Пишешь боту название товара — получаешь топ-10 результатов прямо в чате, и можешь скачать CSV одной кнопкой.

**Пример:**

```
Ты:  iPhone 16 Pro 128gb

Бот: 🔍 iPhone 16 Pro 128gb — найдено 10 товаров

     1. Apple iPhone 16 Pro 128GB
        💰 89 990 ₽  ~~99 990 ₽~~
        ⭐ 4.9 · 3 241 отз.
        Открыть на WB

     2. ...

     [📥 Скачать CSV]
```

### Запуск бота

1. Создай бота через [@BotFather](https://t.me/BotFather), получи токен.
2. Установить зависимости:
   ```bash
   pip install -r requirements.txt
   ```
3. Запустить:
   ```bash
   BOT_TOKEN=<твой_токен> python bot.py
   ```

## CLI

```bash
# Базовый запрос (3 страницы ≈ 300 товаров)
python parser.py "ноутбук"

# Указать количество страниц (100 товаров на страницу)
python parser.py "кроссовки" -p 5

# Указать имя выходного файла
python parser.py "смартфон" -p 2 -o phones.csv
```

### Аргументы

| Аргумент | Описание | По умолчанию |
|---|---|---|
| `query` | Поисковый запрос | обязательный |
| `-p`, `--pages` | Количество страниц | `3` |
| `-o`, `--output` | Имя CSV файла | `query_дата.csv` |

---

## Структура CSV

| Колонка | Описание |
|---|---|
| `название` | Название товара |
| `бренд` | Бренд |
| `цена` | Цена со скидкой, руб. |
| `цена_до_скидки` | Оригинальная цена, руб. |
| `рейтинг` | Рейтинг товара (0–5) |
| `отзывы` | Количество отзывов |
| `ссылка` | Прямая ссылка на товар |
