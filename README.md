# Wildberries Price Parser

Парсер цен с [Wildberries.ru](https://wildberries.ru) по поисковому запросу. Собирает название, бренд, цену, рейтинг и ссылку на товар — сохраняет в CSV.

## Пример вывода

```
🔍 Запрос: «наушники» | Страниц: 3
  Страница 1/3... получено 100 товаров (всего: 100)
  Страница 2/3... получено 100 товаров (всего: 200)
  Страница 3/3... получено 99 товаров (всего: 299)

✅ Сохранено 299 товаров → наушники_20260426_1200.csv
```

Фрагмент CSV:

| название | бренд | цена | цена_до_скидки | рейтинг | отзывы | ссылка |
|---|---|---|---|---|---|---|
| Наушники с микрофоном Jack 3,5 | GQbox | 228 | 972 | 4.8 | 118983 | [ссылка](https://www.wildberries.ru/catalog/16530335/detail.aspx) |
| Наушники проводные с микрофоном | | 87 | 200 | 4.7 | 6641 | [ссылка](https://www.wildberries.ru/catalog/251572603/detail.aspx) |

## Технологии

| Библиотека | Назначение |
|---|---|
| [requests](https://docs.python-requests.org/) | HTTP-запросы к Wildberries API |
| csv | Запись результатов (встроен в Python) |
| argparse | CLI-интерфейс (встроен в Python) |

Сторонняя зависимость только одна — `requests`.

## Установка

```bash
pip install -r requirements.txt
```

## Использование

```bash
# Базовый запрос (3 страницы, имя файла генерируется автом��тически)
python parser.py "ноутбук"

# Указать количество страниц (100 товаров на страницу)
python parser.py "кроссовки" -p 5

# Указать имя в��ходного файла
python parser.py "смартфон" -p 2 -o phones.csv
```

### Аргументы

| Аргумент | Описание | По умолчанию |
|---|---|---|
| `query` | Поисковый запрос | обязательный |
| `-p`, `--pages` | Количество страниц | `3` |
| `-o`, `--output` | Имя CSV файла | `query_дата.csv` |

## Структура CSV

| Колонка | Описание |
|---|---|
| `название` | Название товара |
| `бренд` | Бренд |
| `цена` | Цена со скидкой, руб. |
| `цена_до_скидки` | Оригинальная цена, руб. |
| `рейтинг` | Рейтинг товара (0–5) |
| `отзывы` | Количество отзывов |
| `ссылка` | Прямая ссылка на товар |
